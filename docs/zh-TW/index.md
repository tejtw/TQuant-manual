# TQuant Lab：一站式量化研究平台

!!! info
    歡迎來到 TQuant Lab！本平台整合了全台最完善的量化資料、最強大的事件驅動型回測引擎，以及專業的績效分析工具。無論您是量化研究新手或資深專家，TQuant Lab 都能助您將策略想法轉化為實戰成果。

TQuant Lab 具備三大核心優勢：

-   **卓越數據品質**：整合龐大且經嚴格驗證的 PIT（Point In Time）資料庫，確保回測結果真實可靠。
-   **強大回測引擎**：採用 Zipline 事件驅動型回測框架，支援複雜策略邏輯、模組化建構與高效模擬。
-   **全方位分析工具**：內建 Pyfolio 與 Alphalens 等專業套件，提供詳細績效指標與視覺化報告，助您精準評估策略表現。
