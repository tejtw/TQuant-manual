# 網格布局

Material for MkDocs 使得將區塊排列成網格變得簡單，可以將具有相似意義或相等重要性的區塊進行分組。網格布局非常適合用來建立索引頁面，展示文件中某一大區段的簡要概覽。

欲了解更多詳細資訊，請參閱官方文件：[Grids 詳細說明](https://squidfunk.github.io/mkdocs-material/reference/grids/)
